# QO-Budget 青橙记账
青橙记账是一款极简的记账应用。

是一个基于Vue 全家桶(**Vue 2** + **Vue Router** + **Vuex**) + **TypeScript** 的移动端单页面应用。

数据存储在LocalStorage。

源代码几乎完全使用 **TypeScript实现**，用到了装饰器语法。

编写期间遇到了很多问题，学习到了很多东西。

一些问题的解决过程、学习笔记、反思与总结记录在[本人的博客](https://www.yuque.com/icynescafe/kf5gqy)。
